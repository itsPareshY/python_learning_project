# Python Learning Project

## Installation Instructions

1. Install Jupyter Notebook by running the following command:

   ```bash
   pip install jupyter
   ```

2. Launch Jupyter Notebook using the command:

   ```bash
   jupyter notebook
   ```

3. A new browser window will open, allowing you to create and manage notebooks.

4. In the browser, navigate to your project directory and find the installation_guide.ipynb file.
5. Click on the notebook to open it.
   - To execute a command:
   - Click on a cell to select it.
   - Type your command (e.g., pip install -r requirements.txt).
   - Press Shift + Enter to run the command in that cell.
6. You can add more cells as needed to continue documenting your installation steps or testing code.

## Getting Started

1. Clone the repository or download the project files.
2. Install the required dependencies listed in `requirements.txt`. - refer installation guide jupyter notebook
3. Start exploring Python concepts through the provided examples and exercises.

## Requirements

- Python 3.x
- Flask 2.0.1  
- requests 2.25.1
